# Test Case - Tela de Catálogo de Produtos

| Campo               | Descrição                                                                                      |
| ------------------- | ---------------------------------------------------------------------------------------------- |
| Test Case ID        | TC-003                                                                                         |
| Requisito Funcional | RF-002 – Funcionalidade da seção Products                                                      |
| Tipo de Teste       | Funcional                                                                                      |
| Subtipo de Teste    | Teste de Aceitação                                                                             |
| Objetivo do Teste   | Verificar se a seção Products exibe corretamente as informações dos produtos ao usuário final. |
| Pré-condição        | O usuário deve estar autenticado no sistema.                                                   |
| Dados Necessários   | Possuir um usuário cadastrado e válido para acesso ao sistema.                                 |

## Cenário de Teste

| ID | TC-001                                                                                                                                                                                                                                                                                                             |
|----|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Título | Validar nomes e descrições dos produtos na tela Products                                                                                                                                                                                                                                                           |
| Pré-condições | O usuário deve possuir cadastro no sistema e estar autenticado                                                                                                                                                                                                                                                     |
| Passos | 1. Acessar https://www.saucedemo.com/ <br> 2. Inserir usuário válido <br> 3. Inserir senha válida <br> 4. Clicar em "Login" <br> 5. Localizar o produto "Sauce Labs Backpack" <br> 6. Verificar a descrição do produto <br> 7. Localizar o produto "Test.allTheThings() T-Shirt (Red)" <br> 8. Verificar o nome do produto |
| Resultado esperado | A descrição do produto "Sauce Labs Backpack" deve apresentar texto compreensível ao usuário final, e o produto "Test.allTheThings() T-Shirt (Red)" deve apresentar nomenclatura comercial adequada                                                                                                                 |
| Status | Falhou                                                                                                                                                                                                                                                                                                             |

| ID | TC-002                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
|----|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Título | Validar funcionamento dos filtros de ordenação na tela Products                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| Pré-condições | O usuário deve possuir cadastro no sistema e estar autenticado                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| Passos | 1. Acessar https://www.saucedemo.com/ <br> 2. Inserir usuário válido <br> 3. Inserir senha válida <br> 4. Clicar em "Login" <br> 5. Clicar no filtro de ordenação da tela Products <br> 6. Selecionar a opção "Name (A to Z)" <br> 7. Verificar a ordenação dos produtos <br> 8. Selecionar a opção "Name (Z to A)" <br> 9. Verificar a ordenação dos produtos <br> 10. Selecionar a opção "Price (low to high)" <br> 11. Verificar a ordenação dos produtos <br> 12. Selecionar a opção "Price (high to low)" <br> 13. Verificar a ordenação dos produtos |
| Resultado esperado | Os produtos devem ser ordenados corretamente conforme o filtro selecionado: nome de A a Z, nome de Z a A, preço do menor para o maior e preço do maior para o menor                                                                                                                                                                                                                                                                                                                                                                                        |
| Status | Passou                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| Observação | Ao clicar diretamente na seta do filtro, o menu não abre; porém o filtro funciona corretamente ao clicar no campo ou nas opções disponíveis |
