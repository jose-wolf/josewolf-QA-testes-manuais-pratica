# Test Case - Tela de Catálogo de Produtos

| Campo               | Descrição                                                                                                                                       |
|---------------------|-------------------------------------------------------------------------------------------------------------------------------------------------|
| Requisito Funcional | RF-02 – Funcionalidade da Seção Products                                                                                                       |
| Tipo de Teste       | Funcional                                                                                                                                       |
| Subtipo de Teste    | Teste de Aceitação                                                                                                                              |
| Objetivo do Teste   | Verificar se a tela Products exibe corretamente as informações dos produtos e se as funcionalidades de filtro, carrinho e menu lateral operam conforme esperado |
| Pré-condição        | Usuário `standard_user` autenticado na aplicação                                                                                               |
| Dados de Teste      | Username: `standard_user` \| Password: `secret_sauce`                                                                                          |

---

## Casos de Teste

| Campo              | Descrição                                                                                                                                                                                                                                                                                                                                                 |
|--------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| ID                 | TC-PRODUTOS-001                                                                                                                                                                                                                                                                                                                                           |
| Título             | Validar exibição de nomes e descrições dos produtos na tela Products                                                                                                                                                                                                                                                                                      |
| Pré-condição       | Usuário `standard_user` autenticado e na tela **Products**                                                                                                                                                                                                                                                                                                |
| Passos             | 1. Acessar `https://www.saucedemo.com/` <br> 2. Preencher o campo **Username** com `standard_user` <br> 3. Preencher o campo **Password** com `secret_sauce` <br> 4. Clicar em **Login** <br> 5. Localizar o produto **Sauce Labs Backpack** <br> 6. Verificar a exibição da descrição do produto <br> 7. Localizar o produto **Test.allTheThings() T-Shirt (Red)** <br> 8. Verificar a exibição do nome do produto |
| Resultado Esperado | Os produtos devem exibir nome e descrição completos, sem truncamento, quebra visual ou caracteres inesperados, conforme o catálogo da aplicação                                                                                                                                                                                                           |

| ID                 | TC-PRODUTOS-002                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| Título             | Validar funcionamento dos filtros de ordenação na tela Products                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| Pré-condição       | Usuário `standard_user` autenticado e na tela **Products**                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| Passos             | 1. Acessar `https://www.saucedemo.com/` e realizar login com `standard_user` <br> 2. Clicar no seletor de filtro de ordenação na tela **Products** <br> 3. Selecionar a opção **Name (A to Z)** e verificar a ordenação <br> 4. Selecionar a opção **Name (Z to A)** e verificar a ordenação <br> 5. Selecionar a opção **Price (low to high)** e verificar a ordenação <br> 6. Selecionar a opção **Price (high to low)** e verificar a ordenação                                                |
| Resultado Esperado | Os produtos devem ser reordenados corretamente conforme o filtro selecionado: alfabeticamente de A a Z, de Z a A, por preço do menor para o maior e do maior para o menor                                                                                                                                                                                                                                                                                                                          |
| Status             | Passou                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| Observação         | Ao clicar diretamente na seta do seletor de filtro, o menu não abre; porém o filtro funciona corretamente ao clicar no campo de texto ou nas opções disponíveis                                                                                                                                                                                                                                                                                                                                    |
